Create Project Folder – Kurzanleitung

Create Project Folder richtet sich an alle, die regelmäßig neue Projektordner anlegen und dabei eine einheitliche Struktur benötigen – etwa in der Schule, im Studium, im Büro oder zur privaten Dokumentation. Die App erstellt auf Knopfdruck eine komplette, nach Dateitypen gegliederte Ordnerstruktur (Audio, Excel, EXE, Imagefiles, PDF, PowerPoint etc.) an einem beliebigen Speicherort.

Im Gegensatz zu herkömmlichen Dateimanagern geht es nicht darum, Dateien einzeln zu verwalten, sondern um schnelles, zuverlässiges Anlegen von standardisierten Projektordnern – sowohl lokal (interner Speicher/SD-Karte) als auch im Netzwerk (z. B. SMB-Freigaben).

Die App ist leichtgewichtig, werbefrei, ohne Cloud-Zwang – sie speichert nichts außerhalb deiner Geräte.

🛠 Schritt-für-Schritt: So funktioniert die App

  1. App starten
    Beim Start siehst du einen Link zur Kurzanleitung sowie zwei Bereiche:
    • Lokaler Speicher / SD-Karte
    • Netzwerk
  2. Lokale Projektordnerstruktur erstellen
    1. Tippe auf „Lokaler Speicher / SD-Karte“.
    2. Wähle den gewünschten Speicherort oder erstelle einen neuen Ordner.
    3. Tippe auf „Projektordner erstellen“.
    4. Gib einen Projektnamen ein → Fertig!
    5. Tippe auf den neuen Ordner, um die automatisch erstellte Struktur zu sehen.
  3. Netzwerkverbindung hinzufügen
    1. Tippe auf „Netzwerk“.
    2. Tippe auf das ➕-Symbol.
    3. Gib folgende Daten ein:
      • Host/IP-Adresse
      • Freigabename
        Das ist der Name der freigegebenen Ordnerstruktur (z. B. „Daten“, „Freigaben“).
        Unter Windows findest du ihn in den Ordner-Eigenschaften unter „Freigabe“. Unter Linux/Samba entspricht er dem Namen in der smb.conf.
      • Benutzername & Passwort
    4. Wenn die Verbindung korrekt ist, wird sie gespeichert.
    5. Wenn sie fehlschlägt, kannst du die Eingabe korrigieren oder abbrechen.
  4. Projektordner im Netzwerk erstellen
    1. Tippe auf eine gespeicherte Verbindung.
    2. Die App zeigt die Netzwerkordner an.
    3. Wähle ein Zielverzeichnis oder lege einen neuen Ordner an.
    4. Tippe auf „Projektordner erstellen“, um die Ordnerstruktur anzulegen.
    5. Du kannst auch Ordner umbenennen, löschen oder neu erstellen.
  5. Netzwerkverbindungen verwalten
    Halte eine Verbindung gedrückt, um sie umzubenennen oder zu löschen.
  6. Impressum, Datenschutz, Lizenzbedingungen
    Tippe auf das ℹ️-Symbol in der oberen App-Leiste.
